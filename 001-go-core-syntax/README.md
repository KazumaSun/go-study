# 001-go-core-syntax

Goの文法と標準的な書き方を最短で固めるステップです。

## ゴール
- 変数、型、制御構文、関数、ポインタの説明を即答できる
- `package` / `import` / `main` の関係を説明できる
- 値渡しと参照的な振る舞い（slice/map/pointer）を説明できる

## 学習トピック
- 変数宣言（`var` / `:=`）、ゼロ値、定数（`iota`）
- 基本型、型変換、カスタム型
- `if` / `for` / `switch` / `defer`
- 関数（複数戻り値、named return、可変長引数）
- ポインタの基礎

## 実装課題（自分で実装）
1. 温度変換CLI（Celsius/Fahrenheit）
2. 文字列解析CLI（引数を受けて統計情報を返す）
3. `defer` の実行順を確認する小プログラム

- `defer` はいつ評価され、いつ実行される？
- なぜGoに`while`がないのに困らない？
